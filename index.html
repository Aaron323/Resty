Resty by beders

Tiny HTTP/REST client for Java.

Resty is a small, convenient API to talk to RESTful services from Java.

Its focus is on simplicity and ease-of-use, often requiring only two lines of code to access RESTful web services.

Basic usage is very simple:
Create a Resty instance, then call one of the content type specific methods to get the contents of an URL.

Here is an example on how to use the geonames web service. It retrieves the JSON object (see json.org for details) and gets the name of a place from the zip code: 


Resty r = new Resty();
String name = r.json("http://ws.geonames.org/postalCodeLookupJSON?"+
    "postalcode=66780&country=DE").get("postalcodes[0].placeName").toString();

Resty supports GET, POST (other verbs to come) for JSON, XML, any text or binary data.
For JSON, a query language similar to JasonPath is provided: r.json(url).get("store.book[price>7 && price<12.999].author");

For XML, XPath is supported, mapping XML fragments to neat little objects like Strings: r.xml(url).get("entry/title/text()", String.class);

Resty supports HTTP Authentication. No need to schlepp around the header for each call, just tell Resty when to use authentication: r.authenticate("http://somewhere/else", "login", "yourpasswd".toCharArray());

Here's another example. Getting a static hybrid map from Google Maps and saving it to disk: 


Resty r = new Resty();
File f = r.bytes("http://maps.google.com/maps/api/staticmap?size=512x512" +
  "&maptype=hybrid&markers=size:mid%7Ccolor:red%7C37.815649,-122.477646&sensor=false").
  save(File.createTempFile("google", ".png"));
More complex examples on the Resty git page.
