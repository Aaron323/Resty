Resty

 
 

You can download the JAR file or get your fix at the project page over at Github.


Maven users may want to include this into their project.


<dependency>

    <groupId>us.monoid.web</groupId>

    <artifactId>resty</artifactId>

    <version>0.3.1</version>

</dependency>


Dependencies: none, so far.


New in  0.3.1:

  1. • Option to ignore SSL certificate errors: Resty.ignoreAllCerts (global switch for now)

  2. • New constructor to specify options: new Resty(Option.timeout(3000)); (sets the socket connect timeout)

  3. • Create your own Options (see Resty.Option.Timeout or Resty.Option.Proxy for example)

  4. • Fixed scala naming issue

  5. • Enhanced syntax for JSON queries

  6. • Bugfixes from my contributors. Thank you!

  7. • Proxy support. Thank you, Gabriel. r.setProxy(...) for object r or new Resty(Option.proxy(...)) to carry proxy settings over when traversing paths

  8. • Convenient location header: new Resty().bytes(url, put(someContent)).location(); // gets Location header as URI


New in version 0.2.0: Support for PUT, DELETE, Support for application/multipart-formdata
